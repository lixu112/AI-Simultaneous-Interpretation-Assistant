# AI 同声传译助手 - 产品需求文档

## 1. 产品概述

AI 同声传译助手是一款实时语音翻译工具，帮助用户观看英语（或其他外语）演讲、技术分享、国际会议或网课时，将音频流实时翻译成中文，以字幕或语音形式呈现，降低语言门槛，提升信息获取效率。

### 核心价值
- 实时翻译：边听边译，几乎无延迟
- 多语言支持：英语、日语、韩语等主流语言
- 修正机制：自动纠正识别和翻译错误
- 灵活输出：字幕模式 + 语音播报双模式

## 2. 核心功能

### 2.1 页面结构
| 页面名称 | 功能描述 |
|---------|---------|
| 主页 | 音频源选择、控制面板、实时字幕显示区 |
| 设置页 | 语言选择、语音配置、显示偏好设置 |

### 2.2 功能模块

#### 2.2.1 音频输入模块
- **麦克风实时输入**：直接采集用户周边声音
- **音频文件输入**：上传 MP3、WAV 等音频文件进行翻译
- **系统音频捕获**：捕获电脑系统播放的音频（可选）

#### 2.2.2 实时翻译模块
- **语音识别（ASR）**：将音频转为文字
- **机器翻译（MT）**：将识别文字翻译成中文
- **流式输出**：翻译结果实时显示，支持打字机效果

#### 2.2.3 字幕显示模块
- **双语字幕**：原文 + 中文对照显示
- **字幕修正**：用户可手动修正识别/翻译错误，修正后自动更新
- **历史记录**：保留本次会话所有翻译记录

#### 2.2.4 语音播报模块
- **TTS 播报**：翻译结果实时语音朗读
- **语速调节**：0.5x - 2.0x 语速可调
- **音量控制**：独立音量调节

#### 2.2.5 自动修正模块
- **语义修正**：基于上下文自动修正前文错误
- **实时更新**：修正结果立即反映在字幕和语音中
- **修正历史**：记录所有修正操作

### 2.3 用户交互流程

```
用户启动应用
    ↓
选择音频输入源（麦克风/文件/系统音频）
    ↓
选择源语言 → 选择目标语言（默认中文）
    ↓
开始翻译 → 实时字幕显示 → 语音播报
    ↓
用户可随时暂停/修正/查看历史
    ↓
结束翻译 → 导出字幕/保存记录（可选）
```

## 3. 用户界面设计

### 3.1 设计风格
- **主题**：深邃科技风格 - 暗色背景配渐变霓虹光效
- **色调**：
  - 主色：#6366F1（靛蓝紫）
  - 辅助色：#22D3EE（青色）
  - 强调色：#F472B6（粉色）
  - 背景色：#0F0F1A（深紫黑）
  - 文字色：#E2E8F0（浅灰白）
- **字体**：Noto Sans SC（中文）+ JetBrains Mono（等宽字幕）
- **布局**：左音频控制，右字幕展示，底部状态栏
- **动效**：霓虹呼吸光效、字幕打字机效果、流光过渡

## 4. 技术可行性

### 4.1 前端技术栈
- **框架**：React 18 + TypeScript
- **构建**：Vite
- **样式**：Tailwind CSS + CSS Variables
- **状态管理**：Zustand
- **语音识别**：Web Speech API / 第三方 ASR 服务
- **语音合成**：Web Speech Synthesis API

### 4.2 第三方服务集成
- **翻译 API**：支持多种翻译服务（可选）
- **语音识别**：支持多种 ASR 服务（可选）

### 4.3 本地运行能力
- 主要功能可离线运行（基础语音识别）
- 翻译准确度依赖网络质量

## 5. 性能指标

- **延迟**：字幕显示延迟 < 2 秒
- **准确率**：语音识别准确率 > 90%（标准发音）
- **并发**：支持连续 2 小时稳定运行
- **资源**：CPU 占用 < 30%，内存 < 500MB
